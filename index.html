前期准备

1. 安装git
2. 安装node.js
    查看node.js版本号
    node -v 
    npm -v
3.安装cnpm
    查看cnpm的版本号
    cnpm -v

安装hexo

创建github repo

打开blog项目，修改_config.yml配置文件

上传到github

绑定个人域名

在blog/source目录下新建名为CNAME的文件，文件中添加自己的域名，保存退出

修改及配置主题

以ayer为例，进入blog目录，执行命令：git clone https://github.com/Shen-Yu/hexo-theme-ayer.git themes/ayer
修改_config.yml,修改主题为：
    theme: ayer
然后执行：
hexo clean
hexo generate
hexo deploy
